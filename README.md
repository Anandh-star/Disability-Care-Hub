# Disability-Care-Hub
A feature-rich Disability Care Hub built with HTML5, CSS3, JavaScript, Bootstrap &amp; JSON. Includes AI Chatbot, smart product recommendations, emergency help, cart, Cash on Delivery, Live QR payments, WhatsApp order alerts, testimonials, and a powerful Admin Dashboard.
# 🌟 Disability Care Hub

A modern, responsive, and user-friendly Disability Care Hub web application built using HTML, CSS, JavaScript, Bootstrap, and JSON.

---

## 📌 Project Overview

Disability Care Hub is designed to provide disability support services through a clean and responsive web interface. Users can explore services, register, log in, manage their cart, place orders, and interact with the platform. An advanced Admin Dashboard allows complete management of products, services, users, reviews, and website settings.
